# MySQL connection

## Create a connection file
`server.js`

## Create package.json
`$ npm init -y`

## Install dependencies
`$ npm install`

* Shortcut

`$ npm i`

## Run file
`$ node server.js`

* Shortcut

`$ node server`

* Nodemon saves time so you don't have to keep restarting server with every change

`$ nodemon server`
